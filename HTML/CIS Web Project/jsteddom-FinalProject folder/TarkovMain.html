Welcome!

Escape From Tarkov is a unique game compared to other first person shooter games and has a steep learning curve. This game can be quite frustrating so this website will provide you all the resources on the basics to give you a great foundation. There will be information on the following:

  • The various maps
  • The many weapons and available parts
  • The different types of ammo
  • The 8 traders
  • The many quests
  • The keys and the what they unlock
  • The hideout and skills
  • The scav raid option

How to buy Escape From Tarkov!

Escape From Tarkov can be purchased through this link. There are numerous purchase options for the game.

  • Standard edition - $50
  • Left behind - $82
  • Prepare for escape - $110
  • Edge of darkness - $150

Each of the upgraded editions (all but standard edition) have bonus content. You will get more gear in the game but the main reasons to get the more expensive editions are the extra stash space and bigger secure container. You will learn later the value of these.
