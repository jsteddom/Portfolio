Compiler Program

My last project in CSE 340 (Principles of Programming) was to create a program that would read in a text file which
would contain a C++ program and compile the code. Using linked list and vectors, I created a program that could
compile C++ programs that contained simple math operations, IF statements, FOR loops, WHILE loops, and SWITCH cases
